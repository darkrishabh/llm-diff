# bench-ai

Single package: **CLI** (`bench-ai`, `bench-ai web`), **Next.js UI** (published `src/app` + assets), and **programmatic API** (`import { runDiff, runSuite, … } from "bench-ai"`).

See the [repository README](https://github.com/darkrishabh/bench-ai#readme) for full documentation.

```bash
npx bench-ai "Your prompt" --models claude,ollama
npx bench-ai run --config examples/smoke.yaml
npx bench-ai web
```
